## 📋 О проекте

LiftForge - это современное веб-приложение для отслеживания силовых тренировок. Приложение помогает атлетам-любителям и intermediate лифтерам отслеживать прогресс, анализировать результаты и достигать новых вершин.

### 🎯 Основные возможности

- **📊 Дашборд** - общая статистика и последние тренировки
- **➕ Создание тренировок** - добавление упражнений, подходов с весом и повторениями
- **📚 Библиотека упражнений** - 14 стандартных упражнений + возможность добавлять свои
- **📜 История тренировок** - просмотр, поиск, сортировка, редактирование и удаление
- **📈 Прогресс и статистика** - графики прогресса, личные рекорды, тепловая карта, серии тренировок
- **👤 Профиль и настройки** - тёмная/светлая тема, экспорт/импорт данных

### 🎨 Особенности

- ✨ Drag & Drop для сортировки упражнений
- 🌙 Тёмная/светлая тема с сохранением выбора
- 📱 Адаптивный дизайн (работает на всех устройствах)
- 💾 Автосохранение в localStorage
- 📊 Визуализация прогресса (графики, тепловая карта)
- 🏆 Автоматическое отслеживание личных рекордов
- 🔥 Подсчёт серии тренировок (streak)
- 📤 Экспорт данных в JSON

## 🚀 Установка и запуск

### Требования
- Node.js 18+
- npm или yarn

### Установка

```bash`
# Клонируйте репозиторий
git clone https://github.com/LesanePC/LiftForge.git

# Перейдите в папку проекта
cd LiftForge

# Установите зависимости
npm install

# Запустите в режиме разработки
npm run dev

Приложение будет доступно по адресу: http://localhost:5173

Сборка для production

npm run build

Собранные файлы будут находиться в папке dist

liftforge/
├── src/
│   ├── app/                 # Настройки приложения
│   │   ├── providers/       # Провайдеры (ThemeProvider)

## 🔧 API и хранилище
Все данные сохраняются в localStorage браузера. Ключи:

liftforge-workouts — тренировки

liftforge-exercises — упражнения

liftforge-ui — настройки интерфейса
